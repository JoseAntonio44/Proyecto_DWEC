Curiosidades sobre HTML

HTML es el lenguaje base de todas las páginas web del mundo. Aquí algunas curiosidades:

  • HTML significa HyperText Markup Language.
  • Se creó en 1991 por Tim Berners-Lee.
  • Las etiquetas no distinguen entre mayúsculas y minúsculas.
  • Hoy usamos HTML5, la versión más moderna.
